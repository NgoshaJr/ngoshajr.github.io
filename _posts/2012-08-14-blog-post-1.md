## Introduction
This post will show the steps to grab data from ERDDAP server using an Xarray Python library, how to work with NetCDF files in Python and how to make some graphs.
There are many scientific data servers available in the world that provide satellite data with their own formatting, downloading, and subsetting protocals. However, this may pose a big challenge for biginners to choose a consistent and reliable server. By Using erddap server may fills the gap and unify the advantages in a single service. Thus, users may request data from remote servers like OpeNDap, OBIS, e.t.c., and obtain in the format that requested. erddap supports different file formats such as html table, ESRI .asc and .csv, Google Earth .kml, OPeNDAP binary, .mat, .nc, ODV .txt, .csv, .tsv, .json, and .xhtml, and even some image formats (.png and .pdf).
